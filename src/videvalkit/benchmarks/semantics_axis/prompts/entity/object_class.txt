Hard rules (read before Step 3)
================================================================
- **P1 (multi-trigger)**: 一个观察若**独立违反** N 个 prompt 要求，对应 N 个子项分别 log（如：主物体类别错 + 次要物体类别错 都 true）。
- **P2 (no synonymous double-count)**: 若子项 A 的证据已被子项 B 完全解释，只 log B。Precedence:

  - `主物体局部变形` > `主物体勉强可识别`
- **P3 (saliency upgrade)**: 全局/持续/显著问题必须 S1 或 S2，不因字段名是 L3 而降级。
- **Insufficient evidence**: 写 "证据不足，无法确认"，不猜。
