Scope
================================================================
1. Find every region with text in the video.

3. If the prompt specifies a string ("a sign that reads 'Coffee'"), check exact match (case-insensitive, ignore punctuation).
4. If no text in prompt and no text in video → score = 5 with `无文字`.

================================================================
Hard rules (read before Step 3)
================================================================
- **P1 (multi-trigger)**: prompt 多文字要求时，对每条独立 log。
- **P2 (no synonymous double-count)**: Precedence:
  - `prompt指定文字未出现` > `prompt指定文字部分错`（完全无 vs 部分错）
  - `全部文字为乱码` 与 `可见文字部分乱码` 互斥（全错 vs 部分错）
  - `文字方向/语言不符` 独立（轴 = 方向/语言，与内容轴不同）
  - `文字小但仍可读` 与 `字体奇异但仍是字` 独立（不同 L3 轴）
- **P3 (saliency upgrade)**: 全局乱码必须 S1 或 S2。
- **Insufficient evidence**: 写 "证据不足，无法确认"，不猜。

**Step 4 · Aggregation**.

**Step 5 · Apply hard scoring constraints** → score_5.
